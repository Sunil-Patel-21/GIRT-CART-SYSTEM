# Gift Delivery - MEAN Stack Application

## Project Structure

This is an Angular frontend for a Gift Delivery application converted from React.

## Installation

```bash
npm install
```

## Development Server

```bash
npm start
```

Navigate to `http://localhost:4200/`

## Build

```bash
npm run build
```

## Backend

Make sure the Node.js backend is running on `http://localhost:4000`

## Features

- Browse gift categories (Flowers, Cakes, Chocolates, Soft Toys, etc.)
- Add gifts to cart
- User authentication (Login/Register)
- Place orders
- Responsive design

## Category Mapping (Food → Gift)

- Salad → Flowers
- Rolls → Cakes
- Deserts → Chocolates
- Sandwich → Soft Toys
- Cake → Personalized Gifts
- Pure Veg → Gift Hampers
- Pasta → Greeting Cards
- Noodles → Luxury Gifts
